## Browser support

**Desktop**: Chrome, Safari, Firefox, Opera, IE9+

## Installation 

```shell
    npm install mixins-react

    or 

    jspm install npm:mixins-react
```

## Usage

```js

    import { addListener } from 'mixins-react';

```
